A **Weather App** I built that shows the weather details for your **current location** or any city you search. <br><br>

## Features

- Fetches your current location and displays the weather automatically.  
- Search weather for any city.  
- Unit conversion between Celsius and Fahrenheit.  
- 7-day weather forecast with daily temperature.  
- Weather icons dynamically mapped according to conditions.
